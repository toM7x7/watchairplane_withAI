リアルタイム音声合成デモ | Aivis Cloud API

Aivis Cloud API のストリーミング音声合成をリアルタイムで体験できるデモページです。
音声の生成と再生を同時並行で行い、生成された部分から順次再生を開始します。高品質な音声生成と応答速度をぜひお試しください。
PC Chrome・Android Chrome・Mac Safari・iOS Safari で動作します（Firefox では動作しません）。
ソースコードはこのページの HTML にすべて含まれていますので、ぜひ実装の参考としてご活用ください。
